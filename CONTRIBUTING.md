# 📝 Contributing to CHE MCP Docs

> This repo is the public technical documentation for CHE MCP. The code lives elsewhere — this is the story.

---

## What You Can Do Here

### Fix or Improve Docs

Spotted a typo? Outdated number? Unclear explanation?

- Open an issue or PR with the fix
- English and Spanish both welcome
- Keep the tone professional but warm

### Suggest a New Section

Missing a topic you'd like to see documented?

- Open an issue with the `documentation` label
- Tell us what you want to learn about
- We'll write it or help you write it

### Translate

- English improvements
- Portuguese translations welcome
- Other languages: open an issue first

---

## What Belongs in the Main Repo

Code contributions (new domains, classifier improvements, tests, bug fixes) belong in the main CHE MCP code repository. 

→ **[che-mcp on GitHub](https://github.com/Albano-schz/che-mcp)**

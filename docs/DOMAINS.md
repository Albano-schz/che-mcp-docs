## ❓ Cómo Contribuir

¿Falta tu sector? ¿Tenés datos para aportar? Cada dominio se puede agregar registrando:

1. La fuente de datos (API, CKAN, scraping)
2. Las keywords en español que activan el dominio
3. El handler de datos en `packages/che-pro/src/data-pipeline.ts`

→ [Guía de contribución](../CONTRIBUTING.md)
→ [Prompts de ejemplo](../examples/prompts.md)
